 To run the project you should have node installed on your system and mySQL Server 8.0
 
You can also seed the database initially (optional) so that database will have prefilled entries
-To seed the database navigate to project folder using bash or windows cmd
-Type 'node seeds.js' after entries stop press Ctrl+c
-You can repeat above 2 steps for more prefilled values into the database

-First open app.js file and replace given usrname and password with your mySQL username and pwd
-To start first go to the directory where the entire project resides using windows cmd or bash, 
-then type ‘nodemon app.js’
-this will start the server on port 3000
-and finally open your browser and go to ‘localhost:3000
